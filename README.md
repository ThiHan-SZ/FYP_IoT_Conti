| [Testcase Files](../main/FYP_NextGenIoT_Simulator/TestcaseFiles) | Test Case .txt files containing some test strings |
| [Test Environment](../main/FYP_NextGenIoT_Simulator/TestEnv)  | Directory containing test scripts |
